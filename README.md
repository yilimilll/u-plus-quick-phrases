# U+ 快捷短语插件 ⚡

> 解决交互设计师写 U+ 文档时重复输入 @职能指令的效率问题，一键插入带颜色标签的常用短语。

## ✨ 功能特性

- **快捷插入** — 点击即将短语插入到文档光标位置，无需手动输入
- **彩色职能标签** — 插入后自动带颜色背景，一眼识别责任方
  - `@程序` 🔵 蓝色 | `@拼接` 🟣 紫色 | `@视觉` 🟢 绿色 | `@策划` 🔴 红色 | `@动效` 🩷 粉色
- **分组管理** — 按职能分组，支持折叠/展开
- **搜索过滤** — 输入关键词即时筛选
- **完全自定义** — 自由添加/编辑/删除短语、分组、标签颜色
- **导入导出** — JSON 格式一键分享给同事

## 📦 安装方法

### 1. 下载插件

点击页面右上角绿色 **Code** 按钮 → **Download ZIP** → 解压到本地任意文件夹

### 2. 加载扩展

**Edge 浏览器：**
1. 地址栏输入 `edge://extensions/`
2. 左下角开启「**开发人员模式**」
3. 点击「**加载解压缩的扩展**」
4. 选择解压后的文件夹

**Chrome 浏览器：**
1. 地址栏输入 `chrome://extensions/`
2. 右上角开启「**开发者模式**」
3. 点击「**加载已解压的扩展程序**」
4. 选择解压后的文件夹

### 3. 开始使用

1. 打开 U+ 文档，在编辑区点击获取光标
2. 点击浏览器工具栏的插件图标
3. 右侧弹出快捷短语面板，点击目标短语即可插入

## 🎨 自定义

- **＋** 添加新短语
- **📁** 添加新分组
- **🎨** 管理职能标签颜色（可新增标签、修改颜色）
- **⚙** 导入/导出数据（JSON 格式，方便分享配置）

## 📋 预置短语

| 分组 | 短语 |
|------|------|
| 视觉相关 | @视觉 制作图标、@视觉 制作banner |
| 拼接相关 | @拼接 多语言超长缩小字号、@拼接 多语言超长换行、@拼接 多语言超长省略 |
| 程序相关 | @程序 自适应跟随在后方、@程序 复用通用功能 |
| 策划相关 | @策划 需要配置、@策划 需要确认文案 |
| 动效相关 | @动效 需要制作动效 |
